Programs

Girls Who Code  

This program is for rising juniors and seniors. They spend 7 weeks in the summer learning to code. Each week covers diffrent topics, ranging from websites to algorithms to animation. The Girls Who Code Clubs Program teaches computer science to 6th-12th grade girls.

Girl Development It  

This program provides affordable programs for adult women ages 18-70 interested in learning web and software development in a judgment-free environment.

Black Girls Code  

This program offers summer camps and workshops.The camps provide five days of hands-on, project-based instruction, where young colored ladies ages 10-17 engage in tech instruction.

Ladies Learning Code  

This program based in Canada provides workshops, camps, and after school programs for 6- to 17-year-old girls wanting to learn to code.

CodeNow 

This program provides workshops for both boys and girls around the US.

Geek Girl  

This program offers conferences which include a full day of hands-on workshops, Help Desk, speakers and demos.

App Camp for Girls  

Girls in grades 6 through 8 take two-week long sessions where they brainstorm, design, code, and pitch their apps.

Hackbright Academy  

This ten-week fellowship for women in San Francisco, CA teaches web development fundamentals like HTML and Javascript, as well as computer science fundamentals.
